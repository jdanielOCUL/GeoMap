GDAL - Geospatial Data Abstraction Library

Select language: [English][Russian][Portuguese]
--- это библиотека для работы с растровыми географическими форматами файлов данных. GDAL распространяется Open Source Geospatial Foundation на условиях лицензии X/MIT , то есть является проектом с открытым исходным кодом. Как библиотека GDAL предоставляет вызывающему приложению единую обобщённую модель данных для всех поддерживаемых форматов файлов данных. Помимо этого в состав GDAL входит набор вспомогательных программ, вызываемых из командной строки, для преобразования и обработки данных. Описание выпуска GDAL/OGR 1.5.1 (март 2008) можно найти на странице новостей.

Связанная с GDAL библиотека OGR (исходные тексты которой поставляются вместе с GDAL) предоставляет сходные возможности для работы с векторными данными. На странице обновлений вы сможете найти описание версии 1.3.1, выпущенной в октябре 2005 г.

Конференция

 FOSS4G 2008 основная ежегодная конференция разработчиков открытого и свободного географического программного обеспечения. Будут представлены доклады и проведены семинары по GDAL/OGR, конференцию посетят многие участники сообщества GDAL/OGR. Это важное событие для всех, кто интересуется GDAL/OGR и другими свободными технологиями в области географии и картографии и хочет познакомиться с участниками этого сообщества.

Список рассылки

Для отслеживания основных событий в разработке GDAL/OGR предназначен небольшой по объёму список рассылки gdal-announce.

Для обсуждения процесса разработки и вопросов по использованию GDAL и связанных с проектом технологий существует список рассылки gdal-dev@lists.osgeo.org. Подписаться и просмотреть архивы рассылки можно на специальной странице. Список рассылки также доступен в режиме только для чтения по протоколу NNTP news://news.gmane.org/gmane.comp.gis.gdal.devel и по HTTP http://news.gmane.org/gmane.comp.gis.gdal.devel.

Многих пользователей и разработчиков GDAL/OGR часто можно найти на IRC канале gdal сервера irc.freenode.net.

Отчёты об ошибках

Об ошибках GDAL можно сообщить, и просмотреть при помощи системы Trac.
